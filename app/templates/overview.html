Overview

This demo site is a working example of the Responsive Components strategy outlined in the article Responsive Components: a Solution to the Container Queries Problem.

View demo source Spread the word

To see the demos in action

  • Choose a demo from the left nav.
  • Drag to resize the "Pinned demo" sidebar on the right.
  • Collapse/expand the the "Pinned demo" sidebar and/or left navigation.
  • You can also "pin" any demo in the right sidebar to compare components side by side.

Once you get a feel for the demos, check out the article for an in-depth explanation of the strategy and implementation.

About the breakpoints

  • Each component on this site defines a set of named breakpoints that coresponds to different container sizes (e.g. SM, MD, LG, etc.).
  • When the container changes size, the component's breakpoint automatically updates!
  • To make is easier to visualize the active breakpoint, components display a badge with the breakpoint name in the top corner. Just like this one!
